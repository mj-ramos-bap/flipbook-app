# FlipBook App - Project Instructions

## Overview
Interactive HTML5 flipbook application for converting PDFs into shareable, embeddable flipbooks with admin management, analytics, and branding.

## Technical Stack
- **Framework**: Next.js 14, React 18, TypeScript
- **Database**: PostgreSQL via Prisma ORM
- **Storage**: Google Cloud Storage (GCS) for PDF files and logos — ADC server-side upload/download (no HMAC keys)
- **Auth**: NextAuth.js v4 with credentials provider
- **PDF Rendering**: pdfjs-dist (client-side canvas rendering)
- **Flipbook Engine**: page-flip npm package (3D page-turn animations, portrait + landscape modes)
- **Charts**: Recharts (AreaChart, BarChart)
- **Animations**: Framer Motion + page-flip library
- **Styling**: Tailwind CSS 3
- **Deployment**: Google Cloud Run (`australia-southeast1`), Docker image via Cloud Build + Artifact Registry

## Route Structure
- `/login` - Admin login
- `/admin` - Dashboard with flipbook grid & stats
- `/admin/upload` - PDF upload with drag-drop
- `/admin/flipbooks/[id]` - Settings, Analytics, Embed tabs
- `/admin/branding` - Logo, colors, loading text
- `/view/[uuid]` - Public flipbook viewer (no auth required)

## Database Tables
- User, Flipbook, FlipbookView, PageView, BrandingSettings

## Key Features
- PDF upload (single-part ≤100MB, multipart >100MB)
- PDF.js client-side rendering with canvas
- page-flip library for 3D page-turn animations
- Double-page mode: `usePortrait:false`, two pages side by side with book-spine shadow
- Single-page mode: `usePortrait:true`, advances 1 page per flip (no skipping), instant transitions
- Mode toggle preserves current page position (PageFlip re-initializes on toggle)
- Thumbnails sidebar, zoom (CSS + re-render), fullscreen (native requestFullscreen), keyboard shortcuts
- Password protection per flipbook
- Domain restriction for embed
- Analytics: views, visitors, duration, page-level, geographic, CSV export
- Embed code generation with customizable dimensions
- Branding: logo, primary/secondary colors, loading text
- Security: disable right-click, prevent text selection, signed PDF URLs
- Analytics accuracy: unique visitors by IP, persistent sessionId in sessionStorage, server-side dedup, 30s heartbeat + sendBeacon for duration
- Animated page flip (800ms curl in double mode, instant in single mode)
- Page turn sound effect (Web Audio API, viewer can mute)
- PDF TOC/outline extraction and navigation sidebar
- PDF text search across all pages
- Fixed bottom toolbar (always visible)
- Single admin user model (registration blocked once any user exists)

## User Preferences
- Single admin user model
- Focus on training materials (50 PDFs, 30-200 pages each)
